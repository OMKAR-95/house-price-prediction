#  House Price Prediction using Machine Learning

## 📌 Project Overview
This project uses Machine Learning to predict house prices based on features like area, number of bedrooms, and bathrooms.

## 🎯 Objective
To build a Linear Regression model that can estimate house prices accurately.

## 🛠️ Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

## 📊 Dataset
The dataset includes:
- Area
- Bedrooms
- Bathrooms
- Price

## ⚙️ How It Works
1. Load dataset
2. Split data into training and testing
3. Train model using Linear Regression
4. Predict house prices
5. Visualize data using graph

## ▶️ How to Run

Install dependencies:
pip install -r requirements.txt

Run project:
python src/model.py

## 📈 Output
- Predicted vs Actual Prices
- Graph showing relationship between area and price

## 📌 Conclusion
The model successfully predicts house prices based on given features. It shows how Machine Learning can be used in real-world problems.
